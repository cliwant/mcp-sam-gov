Quick start

Add one server entry to your MCP client. It runs locally over stdio — nothing to host, nothing to sign up for.

Claude Desktop / Claude Code Cursor · Continue · Codex CLI 
{
  "mcpServers": {
    "sam-gov": {
      "command": "npx",
      "args": ["-y", "@cliwant/mcp-sam-gov"]
    }
  }
}

Then ask your agent, in plain language: “Find active SAM.gov solicitations under NAICS 541512 closing this month.” Prefer to try it first? Run npx -y @cliwant/mcp-sam-gov in a terminal.

Engineered not to lie to your agent

Most tools fail silently — a page size reported as a total, an outage returned as “no results,” a missing value shown as $0. Those quiet lies corrupt an agent’s reasoning. This server holds five honesty invariants, enforced by a 3,400+ assertion fault-injection suite in CI.

  • A total is never a page length — totalAvailable is the real upstream total or an explicit lower bound, never the row count.
  • An outage throws, an empty result doesn’t — a 429/5xx/timeout raises a typed error; only a genuine no-match returns empty.
  • Missing is null, never a fabricated zero — an absent amount is null; a real $0 stays $0.
  • Shape drift throws — an unexpected upstream response is surfaced, never fabricated into a partial result.
  • Provenance & freshness are disclosed — non-government sources, snapshot dates, and client-side filtering are labeled.

Keyless & private by design

The server runs locally and collects nothing.

  • No telemetry, no analytics, no accounts — your queries never reach us.
  • Each tool calls the relevant official government API directly over HTTPS.
  • Optional API keys (Census, FRED, BEA, DOL, and a few others for higher limits) stay in your local environment and are sent only to that source.
