# Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short:
- Be respectful and constructive.
- Critique code, not people.
- Federal-contracting work is high-stakes — let's make this resource trustworthy together.

Report unacceptable behavior by opening a private security advisory at
<https://github.com/cliwant/mcp-sam-gov/security/advisories/new>.
